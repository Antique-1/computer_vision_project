## 1. 프로젝트 주제
* 이미지 식별 기술을 활용해 생활 쓰레기의 종류를 자동으로 판별하고, 분리수거 분류(플라스틱 / 유리 / 캔 / 종이 / 음식물 등)를 제안하는 컴퓨터 비전 기반 분리수거 보조 시스템을 구현하는 것을 목표로 한다
* 실시간 객체 탐지 및 분류 기술을 활용하여 카메라로 찍은 이미지에서 쓰레기를 자동으로 인식하고, 올바른 분리수거 방법을 제시한다

---

## 2. 프로젝트 개발 배경 및 목표
* 개발 배경 
  온라인 주문과 포장재 사용 증가로 인해 분리수거의 중요성이 커졌으나, 올바른 분리배출 방법을 모르는 경우가 많음.  
  AI 기반의 이미지 분석을 통해 사용자가 간단히 사진을 찍는 것만으로 분리수거 가이드를 받을 수 있는 시스템을 만들고자 함.

* 목표
  - 쓰레기 이미지에서 객체를 인식하고 종류(플라스틱, 캔, 종이 등) 분류  
  - 분류 결과에 따라 적절한 분리수거 지침(라벨 제거, 내용물 비우기 등)을 안내  
  - 실시간으로 작동 가능한 경량 모델 구현  
  - 모바일 및 웹 환경에서도 사용 가능한 형태로 배포

## 3. 관련 문헌 및 기술 참고
* [Redmon et al. (2016) — You Only Look Once: Unified, Real-Time Object Detection](https://arxiv.org/abs/1506.02640)  
  * YOLO 시리즈의 기초로, 실시간 객체 인식에 적합한 네트워크 구조 제안
  * 본 프로젝트에서는 YOLOv8 기반 모델을 활용하여 빠른 분리수거 물체 검출을 수행함
  
* [He et al. (2017) — Mask R-CNN](https://arxiv.org/abs/1703.06870)  
  * 인스턴스 분할 기반의 객체 검출로, 쓰레기 종류의 복합 장면에서도 정밀한 분리 가능

## 4. 참고 선행 프로젝트 및 차별점
### 4-1. 선행 프로젝트 사례
* [TrashNet (Stanford)](https://github.com/garythung/trashnet)  
  * 6종류의 쓰레기 이미지를 분류하는 CNN 기반 분리수거 프로젝트
  * 단일 이미지 분류 중심으로, 복수 객체나 장면 인식에는 한계가 있음

### 4-2. 차별점
* 본 프로젝트는 단일 이미지 분류를 넘어 다중 객체 검출 기능을 탑재
* 분리수거 규칙 매핑 기능을 추가하여 지역별 쓰레기 분리 방식에 따라 맞춤형 안내 제공.  
* 해설 기능을 추가해 예:  
  > “플라스틱병 → 라벨 제거 후 내용물 비우고 배출”  
  과 같은 세부 가이드를 자동 제공

## 5.모델 구현 계획 및 방향성

### 5-1. 데이터셋 구성
* AI-HUB 분리배출 데이터셋 활용  
  * 실생활 환경의 다양한 분리수거 이미지 포함  
  * 제공된 JSON 라벨을 자체 스크립트로 YOLO(txt)로 변환  
  * 변환된 txt 파일을 이용해 모델 학습에 직접 이용
    
### 5-2. 모델 학습 단계
#### 5-2.1 모델 선택
* 실험한 모델들  
  * YOLOv8 
  * EfficientDet  
  * MobileNetV3 Detector
    
* 최종 선택: YOLOv8  
  * 빠른 속도  
  * 다양한 환경에서도 높은 정확도 확보
    
#### 5-2.2 모델 학습 파이프라인
* JSON → YOLO TXT 자동 변환  
* YOLO 포맷에 맞게 데이터셋 split
* `data.yaml` 자동 생성  
* YOLOv8n 모델 불러오기 
* 다음 설정으로 학습 실행:
  * epochs = 50  
  * imgsz = 640  
  * batch = 16  
  * optimizer = SGD 
  * pretrained weights = yolov8n.pt  

#### 5-2.3 학습 결과물
* `runs/train_auto/exp/weights/best.pt`  
  → 실제 추론에 사용하는 최종 모델 파일
  
#### 5-2.4 학습 결과 적용

학습된 YOLO 모델은 Streamlit 기반 서비스에서 사용
* 이미지 업로드  
* 객체 검출  
* 검출된 클래스에 대한 분리배출 가이드 자동 제공  


### 5-3. 분리수거 가이드 생성 단계
* 인식된 쓰레기 클래스에 따라 분리수거 규칙 DB 매핑  
* 예:  
  | 쓰레기 종류 | 분리수거 가이드 |
  |--------------|----------------|
  | 플라스틱병 | 내용물 비우고 라벨 제거 후 배출 |
  | 유리병 | 뚜껑 분리 후 색상별로 분리 배출 |
  | 캔 | 내용물 비우고 압착 후 배출 |
  | 종이 | 이물질 제거 후 건조시켜 배출 |



### 5-4. 인터페이스 설계
* Streamlit 기반 웹 인터페이스
  - 이미지 업로드 → 검출 결과 및 분리수거 가이드 출력
