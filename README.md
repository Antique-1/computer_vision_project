# 컴퓨터 비전 기반 분리수거 보조 시스템

---

### 1. 프로젝트 주제
* 이미지 식별 기술을 활용해 생활 쓰레기의 종류를 자동으로 판별하고, 분리수거 분류(플라스틱 / 유리 / 캔 / 종이 / 음식물 등)를 제안하는 **컴퓨터 비전 기반 분리수거 보조 시스템**을 구현하는 것을 목표로 한다.  
* 실시간 객체 탐지(Object Detection) 및 분류(Classification) 기술을 활용하여 카메라로 찍은 이미지에서 쓰레기를 자동으로 인식하고, 올바른 분리수거 방법을 제시한다.

---

### 2. 프로젝트 개발 배경 및 목표
* **개발 배경**  
  온라인 주문과 포장재 사용 증가로 인해 분리수거의 중요성이 커졌으나, 올바른 분리배출 방법을 모르는 경우가 많음.  
  AI 기반의 이미지 분석을 통해 사용자가 **간단히 사진을 찍는 것만으로 분리수거 가이드를 받을 수 있는 시스템**을 만들고자 함.

* **목표**  
  - 쓰레기 이미지에서 객체를 인식하고 종류(플라스틱, 캔, 종이 등) 분류  
  - 분류 결과에 따라 적절한 분리수거 지침(라벨 제거, 내용물 비우기 등)을 안내  
  - 실시간으로 작동 가능한 경량 모델 구현  
  - 모바일 및 웹 환경에서도 사용 가능한 형태로 배포  

---

### 3. 관련 문헌 및 기술 참고
* [Redmon et al. (2016) — You Only Look Once: Unified, Real-Time Object Detection](https://arxiv.org/abs/1506.02640)  
  * YOLO 시리즈의 기초로, 실시간 객체 인식에 적합한 네트워크 구조 제안.  
  * 본 프로젝트에서는 YOLOv8 기반 모델을 활용하여 빠른 분리수거 물체 검출을 수행함.
  
* [He et al. (2017) — Mask R-CNN](https://arxiv.org/abs/1703.06870)  
  * 인스턴스 분할 기반의 객체 검출로, 쓰레기 종류의 복합 장면에서도 정밀한 분리 가능.  

* [MobileNetV3 — Howard et al. (2019)](https://arxiv.org/abs/1905.02244)  
  * 모바일 기기에서도 동작 가능한 경량 신경망 모델.  
  * 본 프로젝트에서는 성능과 속도의 균형을 위해 모바일 배포용으로 적용.

---

### 4. 참고 선행 프로젝트 및 차별점
#### 4-1. 선행 프로젝트 사례
* [TrashNet (Stanford)](https://github.com/garythung/trashnet)  
  * 6종류의 쓰레기 이미지를 분류하는 CNN 기반 분리수거 프로젝트.  
  * 단일 이미지 분류 중심으로, 복수 객체나 장면 인식에는 한계가 있음.

#### 4-2. 차별점
* 본 프로젝트는 **단일 이미지 분류를 넘어 다중 객체 검출(Object Detection)** 기능을 탑재.  
* **분리수거 규칙 매핑 기능**을 추가하여 지역별 쓰레기 분리 방식에 따라 맞춤형 안내 제공.  
* **해설 기능**을 추가해 예:  
  > “플라스틱병 → 라벨 제거 후 내용물 비우고 배출”  
  과 같은 세부 가이드를 자동 제공.  
* **실시간 검출** 및 **모바일 경량화 모델(TFLite, ONNX)** 지원으로 실용성 강화.

---

### 모델 구현 계획 및 방향성

### 1. 데이터 수집 및 전처리
* **데이터셋 구성**
  - TrashNet, TACO 등 공개 데이터셋 활용 + 자체 촬영 이미지 추가  
  - 클래스 예시: 플라스틱병, 유리병, 캔, 종이컵, 종이박스, 음식물, 비닐 등  
* **전처리**
  - 이미지 리사이즈, 노이즈 제거, 색상 정규화  
  - 증강(Augmentation): 회전, 밝기 조절, 배경 변경, Mosaic 적용  

---

### 2. 모델 학습 단계
* **모델 선택**
  - YOLOv8, EfficientDet, MobileNet 기반 분류 모델 중 비교 후 채택  
* **학습 전략**
  - GPU 환경에서 전이학습(Transfer Learning) 수행  
  - 성능 평가 지표: mAP, Precision, Recall, F1-Score  
  - 불균형 클래스 해결을 위한 Oversampling 및 Weighted Loss 적용

---

### 3. 분리수거 가이드 생성 단계
* 인식된 쓰레기 클래스에 따라 **분리수거 규칙 DB** 매핑  
* 예:  
  | 쓰레기 종류 | 분리수거 가이드 |
  |--------------|----------------|
  | 플라스틱병 | 내용물 비우고 라벨 제거 후 배출 |
  | 유리병 | 뚜껑 분리 후 색상별로 분리 배출 |
  | 캔 | 내용물 비우고 압착 후 배출 |
  | 종이 | 이물질 제거 후 건조시켜 배출 |

---

### 4. 인터페이스 설계
* **Streamlit 기반 웹 인터페이스**  
  - 이미지 업로드 → 검출 결과 및 분리수거 가이드 출력  
* **모바일 버전(선택사항)**  
  - 카메라 실시간 탐지 및 안내 기능  
  - TensorFlow Lite / ONNX Runtime 기반
